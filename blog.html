Important FAQ

What is the Document object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an Element from the DOM?

You can select an element from the DOM using methods like document.getElementById("id") for an element with a specific ID, document.querySelector("selector") for the first element matching a CSS selector, or document.querySelectorAll("selector") for all elements matching a CSS selecto

What is event delegation in the context of the DOM,& why it is useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements. This is useful because it reduces the number of event listeners needed, improving performance and making the code easier to manage
